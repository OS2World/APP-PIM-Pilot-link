NAME
reminders - Exports your Palm Datebook database into a remind(1) data file format.

CONTENTS

Section
Synopsis
Description
Options
Author
Copyright

SECTION
pilot-link: Userland Conduits

SYNOPSIS
reminders [-p|--port <port>] [--version] [-?|--help] [--usage] [-w|--write]

DESCRIPTION
Your Datebook database will be written to STDOUT as it is converted unless redirected to a file.

OPTIONS

    REMINDERS OPTIONS
-w, --write
Write out the Datebook (not Calendar) database.

    CONDUIT OPTIONS
-p, --port port
Use device file port to communicate with the Palm handheld. If this is not specified, reminders will look for the $PILOTPORT environment variable. If neither are found or supplied, reminders will print the usage information.

-v, --version

Display version information of reminders .

    HELP OPTIONS
-h, --help
Display the help synopsis for reminders and exit without connecting.

--usage

Display a brief usage message and exit without connecting.
